# [seedle](https://seedletimer.nl/)
A timer app to help get productive using the pomodoro method combined with gamification.

# future
- adding a user profile
- creating a logo
- adding a garden
- adding flowers
- receiving flower seeds at completion of a focus session
- planing flower seeds in the garden
- letting flowers grow after each completed focus session
